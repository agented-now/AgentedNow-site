## Naming Checklist

Before committing to a name, verify:

- [ ] `.ai` domain available (use [Namecheap](https://www.namecheap.com), [Porkbun](https://porkbun.com), or [iwantmyname](https://iwantmyname.com))
- [ ] `.com` domain available or affordable
- [ ] Twitter/X handle available (or close variant)
- [ ] LinkedIn company page name available
- [ ] GitHub org name available
- [ ] No trademark conflicts (search [USPTO](https://tmsearch.uspto.gov) and [WIPO](https://branddb.wipo.int))
- [ ] Easy to say in English and Hebrew
- [ ] Easy to spell after hearing it once
- [ ] Looks good in ALL CAPS, lowercase, and Title Case
- [ ] No unfortunate meanings in other languages
