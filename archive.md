* Site supports two user experiences:
    * Traditional: static, visitors can simply read the site context on the right main panel.
    * Agentic: the current visitor can chat with the agent on the left panel, and the site will be adjusted dynamically by the agent based on the chat! The agent (or site pop-up) will inform the visitor that the site can be personalized for them, and invite to chat, and specifically tell the name, company, and intent. The agent will use provided info to web search name, company, title, etc - and adjust a few customizable site sections, in real time. Agent thinking, tool use etc will be shown in the chat - to impress & demonstrate our skills in live. Agent can also answer questions and provide more info on us.
